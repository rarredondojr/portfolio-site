Quote Machine

Quote machine is a project that generates a random quote every time the "New Quote" button is clicked. You can also share the quote since it has twitter functionality embedded as well.
Technologies used: HTML5,CSS3, and Javascript

Markdown Previewer

Markdown previewer is a project that lets you see your markdown. Markdown is a simplified html language so to speak and this app lets you preview your markdown. On the right pane(top if on mobile) the markdown is inputted and then the result is generated on the opposite pane in real time!
Technologies used: HTML5,CSS3, and React

Drum Machine

Drum machine is a project that lets you create beats with your keyboard! By pressing the keys designated on the app each one will generate a different sound. Allowing the user to create a beat!
Technologies used: HTML5,CSS3, and React

Javascript Calculator

Javascript calculator is a project that simply is a javascript version of a basic calculator. The calculator allows you multiply divide, subtract, add, and use exponents. Decimal numbers are supported and several restrictions are applied that would cause malfunctions such as using multiple operators or decimals.
Technologies used: HTML5,CSS3, and React

Pomodoro Clock

Pomodoro clock is a project that allows the user to set a break length and session length in minutes. Once the session countdown has reached 0 the break will begin. Once the break countdown reaches 0 then the session starts again etc...
Technologies used: HTML5,CSS3, and React
